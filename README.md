# NotebookLM
Repositorio criado para abrigar o resumo de uma atividade da DIO, sendo solicitado a criação de um NotebookLM.


🏎️ Sobre o Projeto
O objetivo desta aplicação é facilitar a consulta de dados complexos sobre o novo ciclo da F1, utilizando IA para processar regulamentos técnicos, mudanças nos motores e a entrada de novas fabricantes (como a Audi e a parceria Ford-Red Bull).

🔗 Link de Acesso
Para interagir com o assistente e consultar as fontes de dados, acesse o link abaixo:
[
Acessar Instância do NotebookLM - F1 2026](https://notebooklm.google.com/notebook/13060baa-9ef6-49fd-81ad-9d2b101a59d0)

Nota: Certifique-se de estar logado em uma conta Google para visualizar o conteúdo e vai ser necessária permissão para entrar na instância, pois a plataforma não deixa colocar o mesmo de forma pública..

🛠️ Funcionalidades do Notebook
Resumos Técnicos: Explicações sobre a simplificação do MGU-H e o aumento da potência elétrica.

Perguntas e Respostas: Interface de chat para tirar dúvidas específicas sobre o grid de largada e novas regras de aerodinâmica.

Guia de Estudo: Sugestões de tópicos relevantes gerados automaticamente pela ferramenta.

📖 Como utilizar
Clique no link de acesso acima.

No painel esquerdo, você encontrará as fontes (documentos/PDFs) que alimentam a IA.

Utilize o campo de chat na parte inferior para fazer perguntas como: "Quais as principais mudanças nos combustíveis para 2026?" ou "Quais equipes mudarão de fornecedor de motor?".
